GRAPHENE_SIZE_INIT: @_w: the width @_h: the height

Initializes a graphene_size_t with the given sizes when declaring it, e.g.:

|[ graphene_size_t size = GRAPHENE_SIZE_INIT (100.f, 100.f); ]|
