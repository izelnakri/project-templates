104/**
105 * GtkTreeModelIface:
106 * @row_changed: Signal emitted when a row in the model has changed.
107 * @row_inserted: Signal emitted when a new row has been inserted in
108 * the model.
109 * @row_has_child_toggled: Signal emitted when a row has gotten the
110 * first child row or lost its last child row.
111 * @row_deleted: Signal emitted when a row has been deleted.
112 * @rows_reordered: Signal emitted when the children of a node in the
113 * GtkTreeModel have been reordered.
114 * @get_flags: Get `GtkTreeModelFlags` supported by this interface.
115 * @get_n_columns: Get the number of columns supported by the model.
116 * @get_column_type: Get the type of the column.
117 * @get_iter: Sets iter to a valid iterator pointing to path.
118 * @get_path: Gets a newly-created `GtkTreePath` referenced by iter.
119 * @get_value: Initializes and sets value to that at column.
120 * @iter_next: Sets iter to point to the node following it at the
121 * current level.
122 * @iter_previous: Sets iter to point to the previous node at the
123 * current level.
124 * @iter_children: Sets iter to point to the first child of parent.
125 * @iter_has_child: %TRUE if iter has children, %FALSE otherwise.
126 * @iter_n_children: Gets the number of children that iter has.
127 * @iter_nth_child: Sets iter to be the child of parent, using the
128 * given index.
129 * @iter_parent: Sets iter to be the parent of child.
130 * @ref_node: Lets the tree ref the node.
131 * @unref_node: Lets the tree unref the node.
