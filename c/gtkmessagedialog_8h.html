◆ GtkButtonsType

enum GtkButtonsType

GtkButtonsType: @GTK_BUTTONS_NONE: no buttons at all @GTK_BUTTONS_OK: an OK button @GTK_BUTTONS_CLOSE: a Close button @GTK_BUTTONS_CANCEL: a Cancel button @GTK_BUTTONS_YES_NO: Yes and No buttons @GTK_BUTTONS_OK_CANCEL: OK and Cancel buttons

Prebuilt sets of buttons for GtkDialog.

If none of these choices are appropriate, simply use GTK_BUTTONS_NONE and call [metho.nosp@m.d@Gt.nosp@m.k.Dia.nosp@m.log..nosp@m.add_b.nosp@m.utto.nosp@m.ns].

‍Please note that GTK_BUTTONS_OK, GTK_BUTTONS_YES_NO and GTK_BUTTONS_OK_CANCEL are discouraged by the GNOME Human Interface Guidelines.
