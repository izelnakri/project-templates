36/**

41 * Initializes a #graphene_size_t with the given sizes when
42 * declaring it, e.g.:
43 *
44 * |[<!-- language="C" -->
45 * graphene_size_t size = GRAPHENE_SIZE_INIT (100.f, 100.f);
46 * ]|
