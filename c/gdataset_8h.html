#define G_DATALIST_FLAGS_MASK   0x3

G_DATALIST_FLAGS_MASK:

A bitmask that restricts the possible flags passed to g_datalist_set_flags(). Passing a flags value where flags & ~G_DATALIST_FLAGS_MASK != 0 is an error.
