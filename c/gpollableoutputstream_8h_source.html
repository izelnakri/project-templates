39/**
40 * GPollableOutputStreamInterface:
41 * @g_iface: The parent interface.
42 * @can_poll: Checks if the #GPollableOutputStream instance is actually pollable
43 * @is_writable: Checks if the stream is writable
44 * @create_source: Creates a #GSource to poll the stream
45 * @write_nonblocking: Does a non-blocking write or returns
46 * %G_IO_ERROR_WOULD_BLOCK
47 * @writev_nonblocking: Does a vectored non-blocking write, or returns
48 * %G_POLLABLE_RETURN_WOULD_BLOCK
49 *
50 * The interface for pollable output streams.
51 *
52 * The default implementation of @can_poll always returns %TRUE.
53 *
54 * The default implementation of @write_nonblocking calls
55 * g_pollable_output_stream_is_writable(), and then calls
56 * g_output_stream_write() if it returns %TRUE. This means you only
57 * need to override it if it is possible that your @is_writable
58 * implementation may return %TRUE when the stream is not actually
59 * writable.
60 *
61 * The default implementation of @writev_nonblocking calls
62 * g_pollable_output_stream_write_nonblocking() for each vector, and converts
63 * its return value and error (if set) to a #GPollableReturn. You should
64 * override this where possible to avoid having to allocate a #GError to return
65 * %G_IO_ERROR_WOULD_BLOCK.
66 *
67 * Since: 2.28
68 */
