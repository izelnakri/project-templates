◆ gdk_pixbuf_major_version

GDK_PIXBUF_VAR const guint gdk_pixbuf_major_version

gdk_pixbuf_major_version:

The major version number of the gdk-pixbuf library. (e.g. in gdk-pixbuf version 1.2.5 this is 1.)

This variable is in the library, so represents the gdk-pixbuf library you have linked against. Contrast with the GDK_PIXBUF_MAJOR macro, which represents the major version of the gdk-pixbuf headers you have included. gdk_pixbuf_minor_version:

The minor version number of the gdk-pixbuf library. (e.g. in gdk-pixbuf version 1.2.5 this is 2.)

This variable is in the library, so represents the gdk-pixbuf library you have linked against. Contrast with the GDK_PIXBUF_MINOR macro, which represents the minor version of the gdk-pixbuf headers you have included. gdk_pixbuf_micro_version:

The micro version number of the gdk-pixbuf library. (e.g. in gdk-pixbuf version 1.2.5 this is 5.)

This variable is in the library, so represents the gdk-pixbuf library you have linked against. Contrast with the GDK_PIXBUF_MICRO macro, which represents the micro version of the gdk-pixbuf headers you have included. gdk_pixbuf_version:

Contains the full version of the gdk-pixbuf library as a string. This is the version currently in use by a running program.
